# Task 1 – Data Exploration and Summary using Amazon Sales Dataset

## Project Overview

This project focuses on performing exploratory data analysis (EDA) on an Amazon Sales dataset using Python. The main objective of this task is to understand the structure of the dataset, identify important patterns, handle missing values, and create visualizations that help in better understanding the data.

Data exploration is one of the most important steps in data analytics because it helps in understanding the quality and behavior of the dataset before performing advanced analysis or machine learning.

---

# Objectives of the Project

The main goals of this project are:

* To load and explore the Amazon sales dataset
* To understand the structure and size of the dataset
* To identify different data types
* To check for missing and duplicate values
* To perform basic data cleaning
* To generate summary statistics
* To create meaningful visualizations for better understanding of the data

---

# Tools and Technologies Used

| Tool / Library | Purpose                        |

| Python         | Programming Language           |
| Pandas         | Data Manipulation and Analysis |
| NumPy          | Numerical Operations           |
| Matplotlib     | Data Visualization             |
| Seaborn        | Statistical Graphs and Charts  |

---

# Dataset Description

The dataset used in this project is an Amazon Sales dataset stored in CSV format. It contains sales-related information that can be analyzed to understand trends, distributions, and patterns within the data.

The dataset includes both numerical and categorical columns which makes it suitable for exploratory data analysis.

---

# Steps Performed in the Project

## 1. Importing Required Libraries

The first step was importing all the required Python libraries such as pandas, numpy, matplotlib, and seaborn for handling data and creating visualizations.

---

## 2. Loading the Dataset

The dataset was loaded into Python using the pandas library. After loading the data, the first few rows were displayed to understand the structure and available columns.

---

## 3. Exploring Dataset Structure

Basic information about the dataset was analyzed, including:

* Number of rows and columns
* Column names
* Data types of columns
* Overall structure of the dataset

This helped in understanding how the data is organized.

---

## 4. Summary Statistics

Statistical analysis was performed on numerical columns to calculate:

* Mean
* Median
* Minimum value
* Maximum value
* Standard deviation

These statistics helped in understanding the distribution and behavior of numerical data.

---

## 5. Identifying Missing Values

The dataset was checked for missing or null values. Missing numerical values were filled using median values, while missing categorical values were handled using mode values.

This step improved the overall quality of the dataset.

---

## 6. Removing Duplicate Records

Duplicate rows were identified and removed to ensure cleaner and more reliable data analysis.

---

## 7. Data Visualization

Different types of visualizations were created to better understand the dataset.

### Histograms

Histograms were used to understand the distribution of numerical columns.

### Correlation Heatmap

A heatmap was generated to identify relationships between numerical variables.

### Bar Charts

Bar charts were used to visualize categorical distributions and identify the most common categories.

All visualizations were saved inside the images folder for future reference.

---

# Key Observations

* The dataset contains both numerical and categorical data.
* Some missing values were present and handled successfully.
* Duplicate records were removed during cleaning.
* Numerical features showed different distribution patterns.
* Correlation analysis helped in identifying relationships between variables.
* Visualizations provided a better understanding of trends and category distributions.
